Лицензии «Склада 15»
====================

В зависимости от уровня лицензии «Склад 15» может обладать разными возможностями и функционалом. Рассмотрим подробно каждый из них.

Склад 15 Минимум
----------------

> Работа с маркированным и обычным товаром

**Поддерживается**

* Список операций

  + Приемка
  + Подбор заказа
  + Отгрузка
  + Инвентаризация
  + Списание
  + Возврат
  + Сбор штрихкодов
  + Просмотр справочников
  + Печать штрихкодов
  + Остатки в ячейках
  + Размещение
  + Отбор
  + Перемещение по ячейкам
  + Перемещение по складам
  + Дубликат КМ
* Работа с маркированным товаром в операциях:

  + Приемка
  + Подбор заказа
  + Дубликат КМ
  > Список поддерживаемых маркировок
* Интеграция с конф игурациями 1С
* Бизнес-процессы в конфигурации 1С
* Работа с характеристиками, сериями, серийными номерами, сроками годности, комплектами, транспортными упаковками
* Просмотр справочников, просмотр остатков на складах, просмотр остатков в ячейках
* Работа по документам, созданным на ТСД, и выгруженным из учетной системы
* Печать этикеток на принтер и на ПК
* Офлайн обмен документами и справочниками

**Не поддерживается**

* Запрещено изменять логику работы на ТСД
* Запрещено добавлять свои операции на ТСД
* Нельзя работать с торговыми объектами
* Нет коллективной работы с документом
* Нельзя обмениваться данными в онлайн-режиме
