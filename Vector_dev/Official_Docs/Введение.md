Введение
========

**Обмен документами** — это их выгрузка из «1С:Предприятия» в Клеверенс для работы на мобильном устройстве, а также обратная загрузка с полученными результатами обратно.

> Выгрузка/ загрузка документов необходима только при работе в офлайн-р ежиме. При работе в онлайне все документы из 1С отображаются на экране ТСД в режиме реального времени.

Платные услуги настройки и подключения

Как открыть окно обмена
-----------------------

Для выгрузки/загрузки документов откройте обработку «Клеверенса» и нажмите кнопку «Открыть окно обмена документами».

В окне обмена документы для выгрузки из 1С на терминал сбора данных находятся в левой части (1), документы для загрузки в 1С находятся в правой части (2).

Описание кнопок в окне обмена
-----------------------------

1. Список документов, предназначенных для выгрузки из 1С на мобильное устройство.
2. Список документов, предназначенных для загрузки в 1С с мобильного устройства
3. Обновление списка документов.
4. Выделение всех документов в списке галочками.
5. Отмена выделения всех документов списка.
6. Сортировка документов по возрастанию параметра. Для сортировки потребуется нажать на строку параметра в документе, по которому вы хотите отсортировать документы (например, сумма).
7. Сортировка документов по убыванию параметра. Для сортировки потребуется нажать на строку параметра в документе, по которому вы хотите отсортировать документы (например, сумма). Для сброса сортировки нажмите кнопку «Обновить».
8. Установка периода, за который вы хотите просмотреть документы.
9. Настройки отборов документов для ручного обмена.
10. Выгрузка документов из 1С на мобильное устройство. Перед выгрузкой необходимо от метить нужные документы галочками.
11. Загрузка выбранных документов в 1С. Перед выгрузкой необходимо отметить нужные документы галочками.
12. Выгрузка нескольких документов 1С в один документ Mobile SMARTS. Нужные документы отмечаются галочками.
13. Загрузка нескольких документов Mobile SMARTS в один документ 1С. Нужные документы отмечаются галочками.
14. Загрузка выбранных документов в 1С. Перед выгрузкой необходимо отметить нужные документы галочками.
15. Выпадающий список с дополнительными (или дублирующими) действиями с документами и окном обмена.
16. Выбор колонок, которые будут отображаться в окне обмена (заполняются данными из документов).
17. Просмотр выбранного документа. Перед нажатием на данную кнопку нужный документ необходимо отметить галочкой. Просматривать можно только те документы, которые находятся на мобильном устройстве или были выгружены на сервер.
18. Восстановить вывод подсказок в обработке, если ранее вы нажали кнопку «Больше не показывать».
19. Изменение параметров ок на обмена документами.
20. Подсказка, регулирующая действия пользователя (весь список подсказок в обработке).

Как определить статус документа Клеверенс
-----------------------------------------

Во вкладке «Документы Клеверенс» (2) отображаются документы, выгруженные на подключенные к базе устройства. В зависимости от цвета, которым подсвечены строки, можно определить, в каком статусе находится в данный момент документ:

* **Белый (выгружен)** — документ просто выгрузили из 1С на сервер Mobile SMARTS (кнопка 10 на картинке выше), но работу не начинали. Отображается на ТСД только в том случае, если после его выгрузки была нажата кнопка «Обмен с сервером».
* **Красный (завершен, есть расхождения)** — работа с документом на устро йстве окончена (нажата кнопка «Завершить»), но есть при этом документ собран не полностью (расхождения в количестве). Контроль количества включен.
* **Желтый (завершен, не изменен)** — работа с документом на устройстве окончена (нажата кнопка «Завершить»), но есть расхождения в количестве (или же в документ не было внесено никаких изменений). Контроль количества выключен.
* **Синий (в работе)** — документ открыт на мобильном устройстве и находится в работе.
* **Зеленый (завершен)** — работа с документом на устройстве окончена (нажата кнопка «Завершить»), количество товара в документе полностью сошлось. При этом документ мог быть как выгружен из 1С, так и создан на устройстве.

Также текущий статус документа отображен в строке «Состояние».
