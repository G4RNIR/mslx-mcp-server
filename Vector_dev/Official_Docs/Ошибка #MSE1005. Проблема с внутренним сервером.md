Ошибка #MSE1005. Проблема с внутренним сервером
===============================================

Ошибка **#MSE1005** возникает в том случае, если недоступен опубликованный web (http)-сервис, с помощью которого происходит онлайн-обмен данными между «1С: Предприятием» и «Клеверенсом».

**Решение проблемы:**

Убедитесь, что опубликованный сервис недоступен. Для этого введите адрес публикации в строку браузера (например, *http://127.0.0.1/ut114demo/ws/CleverenceWebExtension.1cws*).

> Название расширения (*CleverenceWebExtension*) в адресе регистрозависимое!

Если в браузере появится «Сервис не найден», необходимо:

1. **Проверить адрес публикации.**  
   **Пример:** http://127.0.0.1/ut114demo/ws/CleverenceWebExtension.1cws  
   *127.0.0.1* — должен быть указан ip-адрес вашего сервера, на котором установлен «Клеверенс».  
   *ut114demo* — имя базы 1С, в которую вы установили расширение «Клеверенс».
2. **Проверить публикацию.**  
   Для этого пройдитесь по всем пунктам инструкции, проверьте, установлены ли все нужные галочки, внесены ли в файл default.vrd необходимые изменения и др. После проверки можете переопубликовать сервис: снять галку с указанных пунктов, опубликовать, затем снова проставить и опубликовать. После этого еще раз проверьте публикацию в браузере.
3. **Проверить адрес публикации, указанный в панели управления.**  
   Для этого откройте панель управления для нужной вам базы → «Внешние соединения» → «1С Предприятие версия 8: OneC\_Connector». Проверьте, какой адрес указан в строке «Сервер» (должен совпадать с адресом опубликованного сервиса и адресом, который вы проверяли в браузере). Кроме того, в пункте «Тип соединения» должен быть указан «WebConnector». Сохраните все изменения, если они были внесены, и попробуйте снова подключиться к 1С.

Если перечисленные манипуляции не помогли, обратитесь в техническую поддержку Клеверенса.
