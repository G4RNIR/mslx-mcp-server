Установка «Склада 15» на ПК из шаблона базы
===========================================

В некоторых случаях может возникнут необходимость иметь на ПК несколько баз Mobile SMARTS для одного и того же продукта, каждая из которых подключена к базе товароучетной программы (например, 1С). Решение этого вопроса — в разворачивании параллельных баз, которые будут использоваться независимо друг от друга.

1. Для добавления новой базы откройте менеджер баз, нажмите кнопку «Добавить», а затем выберите пункт «Создать новую базу данных по шаблону».
2. Выберите шаблон приложения для новой базы. В этом окне будет список готовых шаблонов только в том случае, если у вас уже есть установленные на ПК базы Mobile SMARTS.
3. Выберите нужный шаблон и нажмите кнопку «Далее». В открывшемся окне можно внести изменения в наименование базы, изменить папку, в которой она расположена и выбрать для режим работы для базы. Нажмите «ОК».
4. После этого появится окно мастера установки, который поможет вам настроить устанавливаемое приложение.

Результатом этих действий будет база, установленная параллельно уже имеющимся.
