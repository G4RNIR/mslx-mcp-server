#ONLEX007. Отсутствуют данные маркировки
========================================

Описание проблемы
-----------------

На мобильном устройстве (в онлайн-режиме) или в 1С (офлайн-режим) отображается сообщение с кодом ошибки **#ONLEX007**.

**«1С: Предприятие»:**

**ТСД:**

**Код ошибки: #ONLEX007.** *Отсутствуют сохраненные данные проверки и подбора маркированной продукции! Подробности см. в журнале регистрации 1С.*

Для обоих режимов выполняется запись ошибки в журнал регистрации 1С.

Решение проблемы
----------------

**Способ 1.** Откройте форму проверки и подбора маркированной продукции и нажмите «Сохранить промежуточный результаты проверки».

**Способ 2.** В обработке Клеверенса отключите глобальный параметр «Проверять сохраненные данные проверки и подбора маркированной продукции» --> «Нет».
