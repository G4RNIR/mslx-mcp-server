ERP: Управление предприятием 2.4 (2.5)
======================================

Данная конфигурация представляют собой инновационное решение для управления производственной деятельностью многопрофильных предприятий, в том числе с технически сложным производством. С её помощью осуществляется планирование и бюджетирование производства, контроль продаж и закупок, финансовый учет и многое другое.

В связке со «Складом 15» конфигурация «ERP: Управление предприятием 2.4 (2.5)» обладает следующим функционалом:

* работа с обычным товаром;
* работа с алкоголем ЕГАИС
* работа с маркированными товарами ИС МП;
* работа с транспортными упаковками;
* работа с упаковочными листами;
* работа по характеристикам;
* работа с сериями;
* работа с партиями;
* работа со сроками годности;
* работа с серийными номерами;
* работа с весовым и мерным товаром;
* работа со справочными адресными складами;
* работа с динамическими адресными складами;
* онлайн-обмен с учетной системой.
