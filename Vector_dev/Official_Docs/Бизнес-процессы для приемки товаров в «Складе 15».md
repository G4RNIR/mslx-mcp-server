Бизнес-процессы для приемки товаров в «Складе 15»
=================================================

В данной статье собраны все бизнес-процессы «Склада 15» в обработке 1С, которые могут использоваться для операции «Приемка». Найдите нужный бизнес-процесс в списке справа или введите его назва ние в поиске на странице.

> Названия некоторых бизнес-процессов для разных конфигураций могут незначительно различаться.

Поступление товаров (на основании «Счета от поставщика»)
--------------------------------------------------------

**Краткая сводная по бизнес-процессу:**

|  |  |
| --- | --- |
| **Наименование** | Поступление товаров (на основании «Счета от поставщика») |
| **Где реализован** | «Склад 15» |
| **Документ-основание** | Документ товароучетной системы «Счет от поставщика» |
| **Документ-результат** | Документ товароучетной системы «Поступление (акты, накладные, УПД)» |
| **Документ Mobile SMARTS** | Приемка |
| **Типовой бизнес-процесс** | Да |
| **Документ загружается в исходный** | Нет |
| **Создается новый документ** | Да |
| **Связь между документами** | Да |
| **Запись документа** | Только при онлайн-обмене |
| **Проведение документа** | Нет |
| **Контроль количества** | Выключен (по умолчанию) |
| **Работа с ячейками** | Нет |
| **Разрешено изменение настроек загрузки** | Да |
| **Разрешено изменение настроек выгрузки** | Да |
| **Можно написать свои правила загрузки кодом 1С** | Да |
| **Выгружаемые табличные части из документа 1С** | Товары |
| **Уровень лицензии продукта, в котором поддерживается данный бизнес-процесс** | Базовый, Расширенный, Омни |
| **Данный бизнес-процесс поддерживается с версии продукта** | 1.3.0.х |
