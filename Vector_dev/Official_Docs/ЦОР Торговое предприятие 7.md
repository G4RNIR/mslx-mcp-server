ЦОР: Торговое предприятие 7
===========================

> Поддерживается с версии 1.3.4.х «Склада 15».

В связке со «Складом 15» конфигурация «ЦОР:Торговое предприятие 7» обладает следующим функционалом:

* работа с обычным товаром;
* работа с алкоголем ЕГАИС;
* работа с маркированными товарами ИС МП;
* работа с транспортными упаковками;
* работа по характеристикам;
* работа с сериями;
* работа с партиями;
* работа со сроками годности;
* работа с серийными номерами;
* работа с весовым и мерным товаром;
* работа с динамическими адресными складами;
* онлайн-обмен с учетной системой.
