Как установить одинаковые настройки на всех ТСД одновременно
============================================================

У «Склада 15» есть набор настроек, которые можно задавать индивидуально на каждом ТСД, где он установлен. Но есть способ, который позволяет одновременно установить одну и ту же настройку на всех ТСД (и для всех пользователей), подключенных к одной базе «Склада 15» на ПК. Это очень удобно при большом количестве подключенных терминалов, так как не нужно искать и изменять настройки на каждом терминале.

Как задать одну настройку на всех ТСД, подключенных к базе
----------------------------------------------------------

1. В менеджере баз выберите нужную базу «Склада 15» и откройте для нее панель управления.
2. В панели раскройте меню «Операции» → «Начальные настройки» → «НачалоРаботыНастройкаТоваров» → «Алгоритм».
3. Откроется алгоритм, в котором будут указан весь список настроек с ТСД, Выберите нужную вам настройку (например, «Использовать серии»), и, если нужно, чтобы эта настройка была включена на всех ТСД, вместо значения false укажите true (для всех операций или только нужных).
4. Сохраните изменения в панели.
5. Нажмите кнопку «Обмен с сервером» и перезапустите «Склад 15» на каждом подключенном ТСД, чтобы изменения применились.

Результат
---------

На каждом терминале, подключенном к базе, установлены настройки из панели управления.
