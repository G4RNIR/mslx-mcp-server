#ONLEX006. Работа через веб-сервис возможна только в json формате
=================================================================

В связи с тем, что работа через веб-сервис возможна только в формате json, но иногда в настройках коннектора ошибочно указывают формат xml, на экране ТСД появляется следующее сообщение об ошибке:

Для устранения данной ошибки необходимо:

* В панели управления Mobile SMARTS проверить формат данных, указанный в настройках коннектора 1С: там должно быть «Auto» либо «Json».
* Запустить основную обработку от имени пользователя, который указан в настройках коннектора к «1С: Предприятию». Также для данного пользователя должна быть выбрана база Mobile SMARTS и подключена интеграционная обработка.
* Убедиться, что в настройках подключения к базе Mobile SMARTS в качестве интерфейса подключения выбран REST API и база Mobile SMARTS доступна с сервера.
* Сохранить изменения и закрыть обработку, после чего перезапустить службу сервера Mobile SMARTS.
