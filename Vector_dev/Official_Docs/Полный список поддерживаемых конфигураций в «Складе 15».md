Полный список поддерживаемых конфигураций в «Складе 15»
=======================================================

В процессе интеграции «Склада 15» с «1С: Предприятием» для каждой конфигурации 1С разрабатываются собственные бизнес-процессы, благодаря которым обмен документами между мобильными устройствами сбора данных и учетной системой происходит по уже описанным алгоритмам.

Также для каждой конфигурации 1С создается специальная интеграционная обработка, которая отвечает за работу конфигурации и работает только с определенными её версиями.
