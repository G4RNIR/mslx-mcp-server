Управление торговлей 11.4 (11.5)
================================

> Схожий список бизнес-процессов в интеграции со «Складом 15» имеют следующие конфигурации 1С:
>
> * Управление торговлей 11.5
> * Управление торговлей для Беларуси 3.3 (поддержка с версии 3.3.4.101)
> * Управление торговлей для Беларуси 3.4 (поддержка с версии 3.4.13.148)

В связке со «Складом 15» конфигурация «Управление торговлей 11.4» обладает следующим функционалом:

* работа с обычным товаром;
* работа с алкоголем ЕГАИС;
* работа с маркированными товарами ИС МП;
* работа с транспортными упаковками;
* работа с упаковочным и листами;
* работа по характеристикам;
* работа с сериями;
* работа с партиями;
* работа со сроками годности;
* работа с серийными номерами;
* работа с весовым и мерным товаром;
* работа со справочными адресными складами;
* работа с динамическими адресными складами;
* онлайн-обмен с учетной системой.
