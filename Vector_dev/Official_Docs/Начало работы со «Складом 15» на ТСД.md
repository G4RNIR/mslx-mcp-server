Установка клиента на устройство
-------------------------------

Перед началом работы со «Складом 15» установите на устройство клиент Клеверенса. Это может быть как приложение «Склад 15» (файл «MobileSMARTS.v3.WH.apk»), так и универсальное приложение Mobile SMARTS (файл «MobileSMARTS.v3.apk»), из которого можно подключиться к базе любого продукта «Клеверенс».

### Склад 15 Win CE

> Упрощенный продукт для старых терминалов на Windows и для максимальной скорости для ТСД с Android (подробнее). Только немаркированный товар.

* Приемка
* Подбор заказа
* Отгрузка
* Инвентаризация
* Списание
* Сбор штрихкодов
* Размещение
* Отбор
* Перемещение по ячейкам
* Перемещение по складам
* Быстрый сбор ШК
* Проверка сканирования

### Склад 15 RFID

> Работа с обычным товаром и маркированным RFID-метками.

* Инвентаризация RFID
* Маркировка RFID
* Отгрузка RFID
* Поиск RFID-метки
* Поступление RFID
* Привязка КМ RFID
