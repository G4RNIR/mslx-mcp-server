Как работать в «Складе 15» с одним документом на нескольких ТСД одновременно
============================================================================

В рамках «Склада 15» пользователи разных ТСД могут получить на свои устройства один документ и работать с ним одновременно. Для этого можно использовать:

* продвинутый функционал коллективной работы (только в онлайне);
* более простой способ.

Коллективная работа с документом
--------------------------------

Коллективная работа подразумевает совместный доступ нескольких пользователей к одному документу, который находится на сервере. При этом все ТСД должны работать с базой в онлайн-режиме, иначе доступа к документу не будет.

Что такое коллективная работа

> Для коллективной работы «Складу 15» необходим уровень лицензии Омни.

Коллективная работа возможна только с готовым документом-заданием из учетной системы. Перед началом работы необходимо настроить документ так, чтобы он стал коллективным, и указать конкретных пользователей, которые будут с ним работать.

Выгрузка одного документа нескольким пользователям
--------------------------------------------------

> Поддерживается в любом уровне лицензии «Склада 15».

Данный функционал подразумевает, что отдельные копии одного документа будут выгружены на ТСД разным пользователям. После окончания работы с документом данные от разных пользователей с ТСД будут загружены обратно в один документ учетной системы.

Чтобы выгрузить документ нескольким пользователям, необходимо:

1. Настроить бизнес-процесс, по которому будет происходить выгрузка документа: в пункте «Кому назначать» установить значение «Спрашивать при выгрузке». Сохранить изменения.
2. Выбрать выгружаемый документ в левой части окна обмена («Документы 1С»), отметить его галочкой и нажать кнопку выгрузки в документы Клеверенс. В открывшемся окне выбрать бизнес-процесс, который был отредактирован ранее, а в следующем окне — двух или более пользователей, которым будет выгружен документ.
3. Выгруженный документ появится в правой части окна обмена («Документы Клеверенс»), а значит — и на подключенных к базе устройствах. В шапке выгруженного документа будет указано, кому он был выгружен (поле «Кому назначено»).

> Подробнее о назначении документа пользователю

После того как работа с документом разными пользователями будет завершена, в списке документов Клеверенс появится несколько копий документа. Для их загрузки в один документ 1С используется кнопка «Загрузить несколько выбранных документов из Клеверенс в 1С».
