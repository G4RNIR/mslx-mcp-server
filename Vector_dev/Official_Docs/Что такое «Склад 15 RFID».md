Что такое «Склад 15 RFID»
=========================

«Склад 15 RFID» — это программный продукт, который позволяет выполнять основные товароучетные операции с товаром, маркированным RFID-метками. Кроме того, в рамках данного продукта поддерживается и основной функционал «Склада 15».

> Для корректной работы «Склада 15 RFID» мобильное устройство должно обязательно иметь встроенную RFID-антенну!

Список операций:

|  |  |
| --- | --- |
| **Операции для товара с метками RFID** | **Операции для обычного товара** |
| - Инвентаризация RFID - Маркировка RFID - Отгрузка RFID - Поиск RFID-метки - Поступление RFID - Привязка КМ RFID | - Приемка   - Подбор заказа - Отгрузка - Инвентаризация - Списание - Возврат - Сбор штрихкодов - Просмотр справочников  - Печать штрихкодов - Остатки в ячейках - Размещение - Отбор - Агрегация - Перемещение по ячейкам - Перемещение по складам - Упаковочный лист - Комплектация |
