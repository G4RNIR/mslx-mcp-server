Выгрузка документов из 1С на ТСД с помощью обработки
====================================================

Платные услуги настройки и подключения

В режиме офлайн
---------------

В офлайн-режиме работы с базой Клеверенса требуется регулярно выгружать документы из обработки 1С на мобильное устройство. Для этого:

> Что такое офлайн-режим

1. В обработке Клеверенс откройте окно обмена справочниками. С помощью кнопки «Отбор документов» выберите типы документов 1C, которые будете выгружать на ТСД.
2. Для отображения списка нужных документов выполните отбор по бизнес-процессам (проставьте галочки только для нужных и задайте глубину отбора в днях (кнопка «Настроено»)).
3. После применения настроенных отборов нужные вам документы отобразятся в левой колонке «Документы 1С». После этого можно выбрать из списка один или несколько документов (галочками) и выгрузить их на ТСД с помощью:

* кнопки «Выгрузить из 1С».
* кнопок и (с помощью последней несколько выбранных документов 1С выгружаются в один документ Клеверенса. Нужные документы отмечаются галочками).

В режиме онлайн
---------------

> Что такое онлайн-режим

В режиме онлайн достаточно настроить параметры отбора документов, и на ТСД в режиме реального времени будут отображаться только те документы, которые подходят под эти параметры.

Проверить документ на ТСД
-------------------------

После выгрузки документа из обработки, нажмите кнопку «Обмен с сервером» на ТСД. Откройте на терминале операцию, которая соответствует типу выгруженного вами документа (например, локумент «Приход на склад» будет выгружен операции «Приемка», а документ «Пересчет товара» — в операции «Инвентаризация»). Найдите выгруженный документ с помощью поиска или вручную. Если он есть на ТСД — значит выгрузка прошла успешно.
