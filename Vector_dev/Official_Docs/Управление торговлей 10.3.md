Управление торговлей 10.3
=========================

> Схожий список бизнес-процессов в интеграции со «Складом 15» имеют следующие конфигурации 1С:
>
> * Управление производственным предприятием 1.3
> * Управление производственным предприятием для Казахстана 1.3
> * Комплексная автоматизация 1.1 (без маркированной продукции)
> * Управление торговым предприятием для Казахстана 2.0 (без маркированной продукции)
> * Управление торговым предприятием для Казахстана 2.2 (без маркированной продукции)
> * Управление торговлей для Казахстана 2.0 (без маркированной продукции)
> * Управление торговым предприятием для Украины 1.2 (без маркированной продукции)

**Управление торговлей 10.3** — комплексное решение, которое позволяет консолидировать, обрабатывать и управлять информацией обо всех бизнес-процессах торговой компании в одной информационной системе, начиная с закупки сырья или материалов, заканчивая отгрузкой и доставкой готовой продукции конечным покупателям (контрагентам).

В связке со «Складом 15» конфигурация «Управление торговлей 10.3» обладает следующим функционалом:

* работа с обычным товаром;
* работа с алкоголем ЕГАИС;
* работа с маркированными товарами ИС МП;
* работа с транспортными упаковками;
* работа по характеристикам;
* работа с сериями;
* работа с партиями;
* работа со сроками годности;
* работа с серийными номерами;
* работа с весовым и мерным товаром;
* работа со справочными адресными складами;
* работа с динамическими адресными складами;
* онлайн-обмен с учетной системой.
