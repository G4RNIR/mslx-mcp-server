Как получить новые операции на ТСД после обновления в облачной базе «Склада 15»
===============================================================================

Интеграция «Склада 15» и «МоегоСклада» постоянно дорабатывается: появляется новый функционал, или новые операции. И если новый функционал обновляется автоматически, то новые операции (типы документов) нужно добавлять в базу вручную. Для этого необходимо:

1. Обновить базу в личном кабинете на сайте Клеверенса (подробнее).
2. Подключиться к облачной базе через менеджер баз (подробнее).
3. Открыть панель управления, чтобы отредактировать конфигурацию базы (подробнее).
4. В настройках групп пользователей поставить галочки на новых типах документов (подробнее).

Как обновить базу
-----------------

В списке баз личного кабинета найдите нужную и нажмите кнопку «Доступно обновление». Если этой кнопки нет, значит база уже была обновлена.

Откроется окно со списком изменений в новой версии. Если в списке указаны новые типы документов, потребуется добавить их в базу вручную (самостоятельно или с помощью техподдержки Клеверенс). Нажмите кнопку «Обновить сейчас», и база будет успешно обновлена.

Как подключиться к облачной базе
--------------------------------

Этот процесс уже был подробно описан в статье «Что делать, если завершенный на ТСД документ не появился в «МоемСкладе»».

Как открыть панель управления
-----------------------------

Открыть панель управления можно с помощью кнопки «Открыть» → «Править конфигурацию».

Как добавить новые типы документов
----------------------------------

В панели управления выберите «Польз ователи и группы» --> «Группы пользователей».

Для всех групп пользователей, который работают с базой, проставьте галочки для новых типов документов (в нашем случае это «Оприходование» и «Списание»). Сохраните изменения.

Итоги
-----

Аналогично список операций меняется и на подключенных к базе ТСД.

|  |  |
| --- | --- |
| **Было** | **Стало** |
|  |  |

Если завершенный в «Складе 15» документ не появился в «МоемСкладе», причина может быть в том, что в нем не были заполнены обязательные поля (склад, контрагент, организация), или отсутствуют остатки при включенном контроле количества. Для того чтобы провер ить эти параметры, необходимо вернуть «зависший» документ обратно на устройство, при необходимости отредактировать его и снова завершить. Если все будет сделано правильно, документ появится в «МоемСкладе».
