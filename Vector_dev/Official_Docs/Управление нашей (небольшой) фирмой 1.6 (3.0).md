Управление нашей (небольшой) фирмой 1.6 (3.0)
=============================================

> Схожий список бизнес-процессов в интеграции со «Складом 15» имеют следующие конфигурации 1С:
>
> * Управление небольшой фирмой для Украины 1.6 (без работы с маркированным товаром)
> * Управление небольшой фирмой для Казахстана 1.6 (без работы с маркированным товаром)
> * Автосервис 1.6
> * Автосервис 3

**«Управление небольшой фирмой 1.6 (3.0)»** — решение для оперативного учета, планирования, управления предприятием — для ИП и организаций малого бизнеса.

* работа с обычным товаром;
* раб ота с алкоголем ЕГАИС;
* работа с маркированными товарами ИС МП;
* работа с транспортными упаковками;
* работа по характеристикам;
* работа с сериями;
* работа с партиями;
* работа со сроками годности;
* работа с серийными номерами;
* работа с весовым и мерным товаром;
* работа со справочными адресными складами (только УНФ 3.0);
* работа с динамическими адресными складами;
* онлайн-обмен с учетной системой.
