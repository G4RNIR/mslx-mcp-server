Бизнес-процессы для производства товаров в «Складе 15»
======================================================

В данной статье собраны все бизнес-процессы «Склада 15» в обработке 1С, которые могут использоваться для операции «Производство». Найдите нужный бизнес-процесс в списке справа или введите его название в поиске на странице.

> Названия некоторых бизнес-процессов для разных конфигураций могут незначительно различаться.

Перемещение в производство (проверка документа на ТСД)
------------------------------------------------------

**Краткая сводная по бизнес-процессу:**

|  |  |
| --- | --- |
| **Наименование** | Перемещение в производство (проверка документа на ТСД) |
| **Где реализован** | «Склад 15» |
| **Документ-основание** | Документ товароучетной системы «Перемещение товаров в производство» |
| **Документ-результат** | Документ товароучетной системы «Перемещение товаров в производство» |
| **Документ Mobile SMARTS** | Перемещение по с кладам |
| **Типовой бизнес-процесс** | Да |
| **Документ загружается в исходный** | Да |
| **Создается новый документ** | Нет |
| **Связь между документами** | Да |
| **Запись документа** | Только при онлайн-обмене |
| **Проведение документа** | Нет |
| **Контроль количества** | Включен (по умолчанию) |
| **Работа с ячейками** | Нет |
| **Разрешено изменение настроек загрузки** | Нет |
| **Разрешено изменение настроек выгрузки** | Нет |
| **Можно написать свои правила загрузки кодом 1С** | Да |
| **Выгружаемые табличные части из документа 1С** | Без задания |
| **Уровень лицензии продукта, в котором поддерживается данный бизнес-процесс** | Базовый, Расширенный, Омни |
| **Данный бизнес-процесс поддерживается с версии продукта** | 1.2.5.х |

**Диаграмма бизнес-процесса**

**Поддержка бизнес-процесса в учетных системах:**
