Что такое бизнес-процесс «Склада 15» в «1С:Предприятии»
=======================================================

В рамках интеграции «Склада 15» и «1С:Предприятиия», **бизнес-процесс** — это совокупность правил, по которым документы выгружаются из учетной системы на ТСД/ загружаются с ТСД обратно. Благодаря им документ попадает в нужную операцию на мобильном устройстве, а данные, полученные на ТСД, заполняют подходящий документ 1С.

Готовые бизнес-процессы «Склада 15» не требуют доработок, так как разработаны с учетом большого разнообразия возможных вариантов работы компаний. Некоторым компаниям может потребоваться незначительные дополнительные настройки, которые легко выполняются штатными средствами.

Каждая конфигурация 1С, интегрированная со «Складом 15», имеет свой список бизнес-процессов.

Как использовать бизнес-процесс
-------------------------------

Выбрать бизнес-процесс можно:

* **на мобильном устройстве**, если вы создали новый документ на устройстве и вам нужно выбрать бизнес-процесс, по которому он будет загружаться в 1С (подробнее);
* **в обработке Клеверенс**, если вы хотите выгрузить документ на ТСД (подробнее).

Какие есть бизнес-процессы
--------------------------

В зависимости от типа используемого документа Клеверенс на ТСД, бизнес-процессы делятся на несколько групп:

* для приемки товаров;
* для размещения товаров;
* для отгрузки товаров;
* для отбора товаров;
* для перемещения товаров;
* для инвентаризации товаров;
* для списания товаров;
* д ля возврата товаров;
* для производства товаров;
* для комплектации товаров;
* для упаковочного листа;
* для маркированного товара;
* для комиссии товаров.
