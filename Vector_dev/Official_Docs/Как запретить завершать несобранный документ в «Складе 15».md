Как запретить завершать несобранный документ в «Складе 15»
==========================================================

Контроль полноты сбора товаров по документу — важное условие для снижения ошибок при приемке и отгрузке. Особенно это актуально на складах с большим товарооборотом, где легко упустить неотсканированные позиции.

Начиная с версии 1.6.2.x, в приложении «Склад 15» появилась возможность запретить завершение документа на терминале сбора данных (ТСД), если не все товары из документа были отсканированы. Это помогает строго соблюдать фактическое соответствие между данными учетной системы и результатами работы на устройстве.

В этой статье показано, как включить эту функцию через глобальные параметры базы на ПК и как она будет работать на ТСД при выполнении операций.

Настройка базы на ПК
--------------------

1. Откройте приложение «Склада 15» (менеджер баз --> «Открыть»).
2. Откройте редактор параметров и создайте новый параметр с помощью кнопки «Добавить» (можно добавить несколько).
3. Для нового глобального параметра укажите следующие значения:

   * *ЗапретитьЗавершениеНесобранногоДокументаПриемка* — наименование. Вместо «Приемка» можно использовать название любого другого документа «Склада 15» (например, «Отгрузка»).
   * *Да/Нет* — тип.
   * *Да* — значение.
4. Сохраните изменения.

Работа на ТСД
-------------

После выполненных настроек нажмите на кнопку «Обмен с сервером» на ТСД, подключенном к этой базе, чтобы внесенные изменения применились на устройстве. После этого в операциях, дл я которых был задан новый параметр, при попытке выйти из не полностью собранного документа, кнопка «Завершить» будет неактивна. Закрыть такой документ можно только с помощью кнопки «Временно выйти», чтобы вернуться к работе с ним позже. Данный функционал применим только к документам, выгруженным из учетной системы, но не к созданным на устройстве!
