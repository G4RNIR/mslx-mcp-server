Бизнес-процессы для комиссии товаров в «Складе 15»
==================================================

В данной статье собраны все бизнес-процессы «Склада 15» в обработке 1С, которые могут использоваться для комиссии товаров. Найдите нужный бизнес-процесс в списке справа или введите его название в поиске на странице.

> Названия некоторых бизнес-процессов для разных конфигураций могут незначительно различаться.

Бизнес-процесс «Передача на комиссию (проверка документа на ТСД)»
-----------------------------------------------------------------

**Краткая сводная по бизнес-процессу:**

| **Наименование** | Передача на комиссию (проверка документа на ТСД) |
| --- | --- |
| **Где реализован** | «Склад 15» |
| **Документ-основание** | Документ товароучетной системы со списком товара к передаче |
| **Документ-результат** | Документ товароучетной системы со списком товара к передаче |
| **Документ Mobile SMARTS** | Отгрузка |
| **Типовой бизнес-процесс** | Да |
| **Документ загружается в исходный** | Да |
| **Создается новый документ** | Нет |
| **Связь между документами** | Да |
| **Запись документа** | Да |
| **Проведение документа** | Да |
| **Контроль количества** | Включен (по умолчанию) |
| **Работа с ячейками** | Нет |
| **Разрешено изменение настроек загрузки** | Да |
| **Разрешено изменение настроек выгрузки** | Да |
| **Можно написать свои правила загрузки кодом 1С** | Да |
| **Выгружаемые табличные части из документа 1С** | Без задания |
| **Уровень лицензии продукта, в котором поддерживается данный бизнес-процесс** | Базовый, Расширенный, Омни |
| **Данный бизнес-процесс поддерживается с версии продукта** | 1.6.3 |

**Диаграмма бизнес-процесса:**

**Поддержка бизнес-процесса в учетных системах:**

| **Уч етная система** | **Документ-основание** | **Документ-результат** |
| --- | --- | --- |
| «Управление торговлей 11.5» | Передача товаров | Передача товаров |
| «Комплексная автоматизация 2.5» | Передача товаров | Передача товаров |
