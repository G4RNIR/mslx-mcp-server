После обновления продукта не запускается основная обработка 1С
==============================================================

Если после обновления продукта на платформе Mobile SMARTS перестала запускаться основная обработка 1С, а «1С: Предприятие» выдает ошибку (см. ниже), необходимо:

1. Отрыть базу продукта с помощью ярлыка на рабочем столе вашего ПК или менеджера баз.
2. Открыть редактор параметров базы продукта.
3. Сбросить все настройки базы с помощью одноименной кнопки и сохранить внесенные изменения.
4. Заново пройти мастера первичной настройки базы продукта.
