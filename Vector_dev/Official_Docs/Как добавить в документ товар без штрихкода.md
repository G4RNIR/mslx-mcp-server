Как добавить в документ товар без штрихкода
===========================================

Если у товара, который вы хотите добави ть в документ, отсутствует штрихкод, можно добавить его в документ из выгруженного на устройство справочника (при работе в онлайн-режиме — прямо из учетной системы). Для этого:

1. Введите 0 в строку сканирования произвольной операции и нажмите «Ввод» («Enter»). Откроется справочник товаров.
2. Можно найти нужный товар в списке вручную, а можно воспользоваться поиском. Поиск осуществляется по наименованию товара (части наименования) или по артикулу (части).
3. После того как товар найден, нажмите на него в списке и введите нужное количество. После этого он отобразится в окне как отсканированный.
