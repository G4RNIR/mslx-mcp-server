Бухгалтерия предприятия 3.0
===========================

Конфигурация **«1С: Бухгалтерия предприятия 3.0»** предназначена для автоматизации бухгалтерского и налогового учета, а также для подготовки регламентированной отчетности в организации.

В связке со «Складом 15» конфигурация «1С: Бухгалтерия предприятия 3.0» обладает следующим функционалом:

* работа с обычным товаром;
* работа с алкоголем ЕГАИС;
* работа с маркированными товарами ИС МП;
* работа с транспортными упаковками;
* работа по характеристикам;
* работа с сериями;
* работа с партиями;
* работа со сроками годности;
* работа с серийными номерами;
* работ а с весовым и мерным товаром;
* работа с динамическими адресными складами;
* онлайн-обмен с учетной системой.
